# Document your library

Template has [Docz](https://www.docz.site/) configuration to handle documentation.

```bash
yarn start:docs # starts Docz in development mode, http://localhost:3000/
yarn build:docs # builds Docz pages to `docz-dist` directory
```

## Write
To document component or module simply create file with `.mdx` extension, more info [here](https://www.docz.site/docs/writing-mdx).

## Deploy
After package is published you can build and host documentation at Github Pages or Netlify, using [this manual](https://www.docz.site/docs/deploying-your-docs).
